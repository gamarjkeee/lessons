transition

CSS – переходи чи CSS – transitions можуть застосовуватися всім елементам і навіть до псевдоэлементам. Використовуються для пожвавлення нашої верстки. Що, у свою чергу, призводить до поліпшення взаємодії з користувачем, як правило шляхом приємної анімованої реакції на його дії. Наприклад – наведення та натискання на кнопку. Також властивості transition можна використовувати для побудови нескладних сценаріїв анімації. Фактично, CSS – переходи забезпечують зміну значень інших властивостей із певною анімацією та сценарієм.

transition-duration

Визначає проміжок часу, протягом якого має здійснюватись перехід. Власне - тут ми вказуємо час за який одне значення якості переходить (анімується) в інше. Властивість не успадковується.
Значення за замовчуванням transition-duration: 0;
transition-duration: 0.3s;
transition-duration: 1s;

transition-property

Містить назву CSS-властивостей, до яких буде використано ефект переходу. Значення властивості може містити як одну властивість, так і список властивостей через кому. Властивість не успадковується.
Значення за замовчуванням transition-property: all;
transition-property:color;
transition-property: transform;
transition-property: transform, color;

transition-delay

Затримка виконання переходу. Дозволяє зробити так, щоб зміна якості відбувалася не миттєво, а з деякою затримкою. Час затримки переходу вказується у секундах або мілісекундах. Властивість не успадковується.
Значення за замовчуванням transition-delay: 0s;
transition-delay: 0.5s;
transition-delay: 2000ms;

transition-timing-function

Задає стиль нашого переходу, тобто якийсь сценарій, за яким здійсняться анімація.
Значення за замовчуванням transition-timing-function: ease;
transition-timing-function: linear;
transition-timing-function: ease-in;
transition-timing-function: ease-out;
transition-timing-function: cubic-bezier(0, 1.02, 0.31, 0.45);
transition-timing-function: cubic-bezier(0.5, -0.26, 0.31, 0.45);

transition-universal-function

Універсальний запис: transition: property duration timing-function delay;
Значення за замовчуванням transition: all 0s ease 0s;
transition: transform 0.6s linear 0.3s;
transition: transform 0.6s linear 0.3s, color 0.6s linear 0s;

animation

Як і CSS властивість плавних переходів transition, про яку ми говорили в попередньому уроці, властивість animation покликана зробити нашу верстку динамічнішою, оживити її для кращої взаємодії з користувачем та створення WOW ефекту. Але на відміну від CSS переходів, створення анімації базується на ключових кадрах @keyframes, які дозволяють автоматично відтворювати та повторювати ефекти протягом заданого часу, а також зупиняти анімацію за певною подією.

animation-name

Визначає список анімацій (ключових кадрів), що застосовуються до елемента. Можна вказати через кому. Причому пріоритет останній запис.

animation-duration

Відповідає за тривалість анімації.

animation-timing-function

Встановлює сценарій розвитку анімації між ключовими кадрами.

animation-iteration-count

Відповідає за повтор відтворення ключових кадрів.

animation-direction

Визначає напрямок та тип відтворення ключових кадрів.

animation-play-state

Дозволяє запускати або зупиняти анімацію за подією.

animation-delay

Відповідає за затримку перед відтворенням.

animation-fill-mode

Визначає, які значення анімованих css властивостей застосовуються до об'єкта після завершення анімації.

universal-function

Універсальний запис animation
